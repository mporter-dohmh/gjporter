500 - Internal Server Error

Oops!

Error code 500 means Internal Server Error. While the name may imply a problem with your web hosting service, this is usually not the case. The most common reason for seeing this error is that you have syntax errors in your .htaccess file or that you have set the incorrect permissions on files inside of your account (or both). If this error recently came up after working on your site, these common reasons are most likely the cause.
